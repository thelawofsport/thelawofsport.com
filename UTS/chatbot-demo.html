AI Study Assistant — Demo

This page demonstrates the chatbot widget for The Law of Sport student portal. Click the green chat button in the bottom-right corner to open the assistant and try some queries.

The prototype runs with pre-built demo responses to show the interaction patterns. When connected to a live AI API, it will generate contextual answers using the system prompt and your full course content.

Try asking about...

  • "What topics does Module 4 cover?" — see how it navigates course content
  • "Where can I learn about anti-doping?" — see the guided discovery pattern
  • "I have an assessment question" — see the Q&A page redirect + guardrails
  • "How do player contracts work?" — see cross-referencing across lectures
  • "Can you give me legal advice about my situation?" — see the boundary response
  • "Tell me about match-fixing" — see how it links related topics
  • "What's the duty of care in sport?" — see negligence/injury topic handling

Response Behaviour

  • Guided discovery: Directional answers, never absolute conclusions
  • Source references: Every answer points to specific lectures/modules
  • Assessment Q&A pages: Assessment queries redirect to your evolving Q&A bank
  • Assessment guardrails: Tighter responses near due dates (configurable)
  • Forum fallback: "Can't find it? Post on the discussion forum"
  • Legal advice boundary: Politely declines real-world legal queries
  • Context-aware: Detects which page the student is on

Integration — How to add to your site

Step 1: Add one line before </body> on any page:

<script src="chatbot-widget.js"></script>

Step 2: For production, set up a backend API proxy. A minimal Node.js example:

// server.js — proxy to Anthropic Claude API
const express = require('express');
const Anthropic = require('@anthropic-ai/sdk');
const app = express();

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

app.post('/api/chat', express.json(), async (req, res) => {
    const response = await client.messages.create({
        model: 'claude-sonnet-4-6',
        max_tokens: 500,
        system: req.body.system,
        messages: req.body.messages
    });
    res.json({ content: response.content[0].text });
});

Step 3: In chatbot-widget.js, set this.isDemo = false and update the apiEndpoint.

Step 4: Configure assessment due dates in the CHATBOT_CONFIG object to activate tighter guardrails near submission deadlines.

Estimated API cost: ~$0.01-0.03 per student query using Claude Sonnet. A class of 80 students asking 10 questions each per week ≈ $8-24/month.

The widget automatically matches dark mode, detects the current page context, and is fully responsive on mobile.
